# 📘 WQU Data Science Lab (Practice Repository)

## Overview
This repository contains lab assignments completed as part of the Applied Data Science Lab at WorldQuant University.

The work consists of multiple guided, end-to-end data science projects covering real-world problems across different domains.

## Purpose
- Practice core data science concepts through hands-on labs
- Work with real datasets and build end-to-end pipelines
- Strengthen understanding of machine learning and time series techniques

## Topics Covered
- Data preprocessing and feature engineering
- Regression and classification models
- Clustering (K-Means, PCA)
- Time series forecasting (ARIMA, GARCH)
- A/B testing and statistical analysis
- Model evaluation and validation

## Projects Included
- Housing price prediction (Mexico & Buenos Aires)
- Air quality forecasting (Nairobi)
- Earthquake damage prediction (Nepal)
- Bankruptcy prediction (Poland)
- Customer segmentation (USA)
- A/B testing analysis
- Volatility forecasting (India)

## Note
This repository is based on guided lab exercises from WorldQuant University.  
It is intended for learning and practice, not as an original standalone project.
