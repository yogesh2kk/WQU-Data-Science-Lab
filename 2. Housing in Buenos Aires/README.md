# 2. Housing in Buenos Aires

## Project Overview

This project focuses on apartment sales in Buenos Aires, Argentina. The goal is to build a linear regression model that predicts apartment prices using property features.

The project includes data cleaning, preprocessing, feature engineering, and model evaluation to improve prediction accuracy and reduce overfitting.

## Skills Demonstrated

- Data cleaning and preprocessing
- Linear regression modeling
- Missing value imputation
- Categorical feature encoding
- Data pipeline creation
- Model evaluation
- Overfitting reduction

## Learning Outcomes

Throughout this project, I learned how to:

- Build a linear regression model to predict apartment prices
- Create a data processing pipeline
- Impute missing values
- Encode categorical variables
- Improve model performance by reducing overfitting
- Evaluate prediction accuracy

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Key Question

**How accurately can apartment prices in Buenos Aires be predicted using property features?**
