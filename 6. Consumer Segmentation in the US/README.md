# 6. Consumer Segmentation in the US

## Project Overview

This project focuses on customer segmentation using unsupervised machine learning techniques. The objective is to identify meaningful groups of U.S. consumers based on their financial characteristics and behaviors.

Using clustering algorithms and dimensionality reduction techniques, the project uncovers patterns within consumer data and visualizes customer segments. The project concludes with the development of an interactive dashboard using Plotly Dash.

## Learning Outcomes

Throughout this project, I learned how to:

* Identify and analyze credit-fearful households
* Compare subgroup characteristics using side-by-side bar charts
* Build and tune K-Means clustering models
* Evaluate clustering performance using inertia and silhouette scores
* Select meaningful features using variance and trimmed variance
* Apply Principal Component Analysis (PCA) for dimensionality reduction
* Visualize clusters in lower-dimensional space
* Interpret cluster centroids and group characteristics
* Design and deploy an interactive dashboard with Plotly Dash
