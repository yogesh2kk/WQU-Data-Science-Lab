# 4. Earthquake Damage in Nepal

## Project Overview

This project focuses on predicting earthquake damage to buildings in Nepal using machine learning classification models. The objective is to identify factors associated with severe building damage and develop models capable of predicting damage levels based on structural, geographic, and socioeconomic characteristics.

The project includes extracting data from a SQLite database, performing feature engineering, building classification models, and evaluating model performance.

* Data extraction and preprocessing

* Classification modeling
* Logistic Regression

* Bias and fairness analysis
