# 5. Bankruptcy in Poland

## Project Overview

This project focuses on predicting corporate bankruptcy using financial data and machine learning classification models. The goal is to identify companies at risk of bankruptcy and develop predictive models that can support risk assessment and decision-making.

The project involves working with compressed JSON datasets, handling imbalanced data, training ensemble learning models, and evaluating performance using business-focused metrics.
