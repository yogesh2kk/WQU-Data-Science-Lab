# 1. Housing in Mexico

## Project Overview

This project explores a dataset of more than 21,000 residential properties in Mexico to investigate whether housing prices are influenced more by property size or location.

Using Python and data analysis techniques, the dataset was cleaned, explored, and visualized to identify patterns and relationships between housing features and property prices.

## Skills Demonstrated

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization with Python
- Correlation analysis
- Feature investigation and interpretation
- Real estate market analysis

## Learning Outcomes

Throughout this project, I learned how to:

- Build a linear regression model using Scikit-Learn
- Create a data processing pipeline
- Handle missing values and encode categorical variables
- Evaluate model performance and identify overfitting
- Interpret the impact of different features on predicted housing prices

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Key Question

**Are housing prices in Mexico influenced more by property size or location?**

The analysis uses statistical techniques and visualizations to explore the relationship between housing characteristics and property values.
